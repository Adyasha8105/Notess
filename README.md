## Setup

  ##### Clone the repository
```bash
git clone https://github.com/Adyasha8105/Notess.git
```
  ##### Move to the desired folder
```bash
cd \Notess
```
  ##### To install the dependencies, simply write
```bash
npm install
```

  ##### To run the app, simply write
```bash
npm start
```
